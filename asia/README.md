## react + typescript 사용하여 웹 페이지 만들기

- 포트폴리오 및 react와 타입스크립트 연습, 아버지 가게 웹페이지 만들기

## 현재 사용된 스택, 라이브러리

- React, Typescript, styled-components, react-router-dom, react-dom

## 앞으로 구현해야할 기능

1. 메인 화면 데이터 나오게 하기
2. Footer
3. 로그인 및 회원가입

5. 장바구니
6. 사이트내에 있는 검색
7. 로고 클릭시 메인화면으로 넘어가기

9. 반응형 웹 페이지로 바꾸기
